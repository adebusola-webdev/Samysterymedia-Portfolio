# Samystery Media Portfolio

A stylish, modern, and immersive portfolio website for Samystery Media, a creative media brand focused on mobile videography and content creation.

## About Samystery Media

**Samystery Media** is a creative media brand founded by **Samuel Ayeni**.
**Role:** Mobile Videographer | Editor | Content Creator
**Position:** CEO & Director
**Location:** Lagos, Nigeria

### What We Do
Samystery Media captures moments from events and turns them into lasting memories through high-quality mobile video and editing. We cover a wide range of events including:
- Weddings
- Corporate events
- Church programs
- Lifestyle content
- Personal brand content
- Social media reels

We are also open to collaborations with brands and creatives.

### Clientele
Individuals, event planners, corporate organizations, churches, lifestyle content creators, and brands.

## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key (if applicable for AI features)
3. Run the app:
   `npm run dev`
